## Vocabulary

- あ
    - あめ (Rain)
    - あい (Love)
    - いくさ (You)

- い 
  -  いいえ (No)
  -  いき (Breath)
  -  いし (Stone)

- う
  - うえ (Up)
  - うち (Home)
  - うたう (Sing)

- え
  - えさ (Feed)
  - えいご (English)
  - えんぴつ (Pencil)

- お
  - おす (Push)
  - おいし (Delicious)
  - おおかみ (Wolf)
